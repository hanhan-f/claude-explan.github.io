Claude code 安装 ​

1. 使用node命令安装 ​

npm install -g @anthropic-ai/claude-code --registry=https://registry.npmmirror.com

2. 安装检查 ​

输入命令 claude -v 得到结果

C:\Users\Admin>claude -v
2.1.19 (Claude Code)

证明安装成功

3. 环境配置 ​

环境变量配置

claude code 依赖于git-bash 运行，设备需要提前安装git-bash，（在我们的设备都安装有）

在系统变量中添加记录值

CLAUDE_CODE_GIT_BASH_PATH = E:\work\Git\bin\bash.exe

4. api key 配置 ​

Claude 官方会直接封中国用户的账号，所以使用官方账号门槛很高。

推荐使用中转方案，或者用国产大模型平替。,此处使用deepSeek进行演示

第一步 ：

  • 打开 DeepSeek 开放平台： https://platform.deepseek.com
  • 注册 DeepSeek 账号并登录

第二步 ：

  • 获取 API： DeepSeek Platform
  • 新建一个 APIkey，名字随便取
  • 复制 API key

第三步 ：在 Claude code 里使用

复制以下所有的内容，把第二行里的 DEEPSEEK_API_KEY 换成你上一步生成的 APIkey，其他不要动：

text
export ANTHROPIC_BASE_URL=https://api.deepseek.com/anthropic
export ANTHROPIC_AUTH_TOKEN=DEEPSEEK_API_KEY
export API_TIMEOUT_MS=600000
export ANTHROPIC_MODEL=deepseek-chat
export ANTHROPIC_SMALL_FAST_MODEL=deepseek-chat
export CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC=1
  • 粘贴到命令行里，然后按回车键（注意！！！！，使用git-bash命令行面板，因为export 是git-bash的命令）
  • 之后，不会有任何反馈，这是正常的
  • 然后输入 claude ，按回车，就可以进入 Claude code 了！（中间会有一些让你确认的，选 yes，继续就 OK）

5. 报错 ​

在配置前使用 claude命令启动时会出现这样的报错

C:\Users\Admin>claude
Claude Code on Windows requires git-bash (https://git-scm.com/downloads/win). If installed but not in PATH, set environment variable pointing to your bash.exe, similar to: CLAUDE_CODE_GIT_BASH_PATH=C:\Program Files\Git\bin\bash.exe

别怀疑，就是忘记添加CLAUDE_CODE_GIT_BASH_PATH环境变量了
